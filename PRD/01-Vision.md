# Vision

## Project

SWGOH Guild Suite is an open-source toolkit for Star Wars: Galaxy of Heroes.

The project originated as an internal utility for the BuzzLighter guild and is primarily developed to simplify guild management and reduce repetitive manual work.

The long-term goal is to provide a single platform that combines planning, analytics and reference tools for both individual players and guild officers.

## Core Goals

- Reduce manual work for players and officers.
- Provide reliable planning tools.
- Offer comprehensive game analytics.
- Keep all modules based on a common data model.
- Support multiple trusted data providers.
- Never require access to a player's game account.

## Key Differences

Compared to existing community tools, SWGOH Guild Suite focuses on:

- Near real-time guild data using Comlink API.
- A unified internal data model shared by every module.
- A graphical interface instead of chat-command driven workflows.
- Modular architecture where new tools reuse existing data.
- Full compliance with EA Terms of Service.
